# ADR-001 — Stack tecnologico e perimetro MVP

**Progetto:** SICYNT Cyber & New Tech Map  
**Versione:** MVP v0.1  
**Stato:** approvato dal tavolo  

## Contesto

Il progetto mira a realizzare una web app pubblica, leggera e a basso costo per visualizzare su mappa e nel tempo gli eventi Cyber e New Tech italiani nel prossimo orizzonte temporale.

## Decisione

Lo stack MVP v0.1 è:

- Frontend: Next.js + React
- Deploy: Vercel
- Mappa: React Leaflet + OpenStreetMap
- Dataset: JSON statico nel repository
- Backend: assente nella v0.1
- Accesso: pubblico, sola visualizzazione
- Pipeline: sviluppo locale → GitHub → deploy automatico Vercel

## Funzioni incluse nella v0.1

- Mappa Italia
- Marker eventi
- Filtro temporale, default 60 giorni
- Filtro categorie
- Scheda evento
- Link ufficiale/fonte
- Dataset manuale `data/events.json`

## Regola temporale

Mostrare gli eventi il cui intervallo `start_date` → `end_date` interseca il range selezionato.

## Categorie iniziali

- Cybersecurity
- AI
- New Tech
- Difesa
- PA
- Università/Ricerca
- Startup/Business
- Crittografia/Crittologia

## Esclusioni esplicite v0.1

- Login
- Admin
- AI
- Crawler
- Community
- Notifiche
- Routing/percorsi
- Google Maps
- Suggerimenti utenti
- Ranking

## Principio guida

Se una funzione non serve a validare l'utilità della mappa, resta fuori dalla v0.1.
